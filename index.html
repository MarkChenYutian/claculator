THE Clac-ulator

How to Use (Read Me!)

  1. Enter some clac/exp expression in the input box, split each operand by space! (i.e. input 1 1 + instead of 11+)
    1. Click "Run" to execute everything at once
    2. Click "Step" to execute a single step of clac
  2. When "Show Trace" is on, you can see the history of stack and queue
  3. When "Show Function" is on, you can see all the function (symbol) that is currently defined in clac
  4. If the queue is empty, you can type new input into the input box to add token to the queue
  5. If you want to reset clac (discard everything), click "Restart"
  6. If Clac Core Exception pops out, RESTART the clac before making any Step/Run

Known Caveats

  1. When operates on large numbers, number will not overflow like C0.
  2. When Clac Core Exception pops out, if one continue to press 'step', weird result will occur.
  3. If anything unexpected happens, feel free to sent me (yutianch@andrew.cmu.edu) an email or open up an issue on This GitHub Repo
